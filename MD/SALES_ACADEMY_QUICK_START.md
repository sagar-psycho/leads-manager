# Sales Academy - Quick Start Guide

## For Super Admin / Admin

### Creating Your First Training Course

1. **Navigate to Sales Academy**
   - Click "Sales Academy" in the sidebar
   - Admin dashboard loads automatically

2. **Create Categories First** (One-time setup)
   - Click "Categories" button
   - Click "Add Category"
   - Enter name: "Sales Techniques"
   - Enter description: "Core sales training courses"
   - Repeat for other categories:
     - Product Knowledge
     - Company Policies
     - Compliance
     - Customer Service

3. **Create a Training Course**
   - Click "New Course" button
   - Fill in course details:
     - **Title**: "Introduction to ABRA Logistics"
     - **Description**: "Learn about our company, mission, and values"
     - **Category**: Select "Company Policies"
     - **Campaign**: Leave as "General Training"
     - **Duration**: 30 minutes
     - **Status**: Draft (for now)
     - **Mandatory**: ✓ Check this box

4. **Add Training Sections**
   - Click "Add Section" button
   - **Section 1 - Video Introduction**:
     - Title: "Welcome to ABRA Logistics"
     - Type: Video
     - Video URL: https://youtube.com/watch?v=...
     - Click "Save Section"
   
   - Click "Add Section" again
   - **Section 2 - Company History**:
     - Title: "Our Story"
     - Type: Text
     - Content: "Founded in 2010, ABRA Logistics..."
     - Click "Save Section"
   
   - Click "Add Section" again
   - **Section 3 - Knowledge Check**:
     - Title: "Test Your Knowledge"
     - Type: Quiz
     - Click "Add Question"
       - Question: "What year was ABRA Logistics founded?"
       - Option 1: "2008"
       - Option 2: "2010"
       - Option 3: "2012"
       - Option 4: "2015"
       - Correct Answer: 2
     - Click "Save Section"

5. **Save as Draft**
   - Click "Create Course"
   - Course saved and appears in management table
   - Status shows "Draft" (yellow badge)

6. **Test the Training**
   - Ask a trusted Sales Member to test it
   - They can access draft courses for testing
   - Get feedback and make edits

7. **Publish the Training**
   - Find course in management table
   - Click "Publish" button (green checkmark)
   - Confirm publication
   - Course now visible to all Sales Members
   - Status shows "Published" (green badge)

### Setting Up Mandatory Onboarding

1. **Create Required Trainings**
   - Create "Company Introduction" (mark as mandatory)
   - Create "CRM Basics" (mark as mandatory)
   - Publish both courses

2. **Test the Flow**
   - Create a test Sales Member account
   - Login as that user
   - Try to access Leads
   - Should see mandatory training warning
   - Complete the trainings
   - Leads should unlock

3. **Monitor Completion**
   - Click "Reports" in Sales Academy
   - Check completion rates
   - Follow up with members who haven't completed

### Creating Campaign-Specific Training

**Option 1: During Campaign Creation**
1. Create new campaign (e.g., "Sea Freight")
2. System prompts: "Create training for this campaign?"
3. Click OK
4. Training Builder opens with pre-filled name
5. Mark as mandatory
6. Add sections about Sea Freight
7. Publish

**Option 2: Manually**
1. Click "New Course"
2. Enter title: "Sea Freight Sales Training"
3. Select Campaign: "Sea Freight"
4. Mark as mandatory
5. Add campaign-specific content
6. Publish

## For Sales Members

### Accessing Sales Academy

1. **Login to CRM**
2. **Click "Sales Academy"** in sidebar
3. **Learning Dashboard** loads showing:
   - My Progress card
   - Available courses
   - Completed courses
   - In-progress courses

### Starting a Training

1. **Find a course** in "Available Courses"
2. **Click "Start Training"** button
3. **Training player** opens fullscreen
4. **Read/watch** the first section
5. **Click "Mark Complete"** at bottom
6. **Next section** loads automatically
7. **Repeat** until all sections completed
8. **Certificate** issued (if configured)
9. **Return to dashboard**

### Completing Mandatory Training

1. **If you see a warning** on login:
   - "Mandatory Training Required"
   - Lists incomplete courses
2. **Click "Go to Sales Academy"**
3. **Complete each listed course**:
   - Start course
   - Complete all sections
   - Mark each section complete
4. **After last course** completed:
   - Leads module unlocks
   - Can start working with leads

### Tracking Progress

1. **Progress Card** shows:
   - Courses started
   - Courses completed
   - Overall completion percentage
2. **Course Cards** show:
   - Green checkmark = Completed
   - Yellow progress bar = In progress
   - Blue "Start" button = Not started
3. **My Certificates**:
   - View earned certificates
   - Download as PDF (future feature)

### Using Knowledge Base

1. **Click on article** to expand
2. **Search** using tags
3. **Reference** during customer calls
4. **Bookmark** useful articles

## Common Tasks

### Editing a Published Course

**⚠️ Warning**: Editing published courses affects active learners

1. Find course in management table
2. Click "Edit" (pencil icon)
3. Make changes
4. Click "Update Course"
5. Changes apply immediately

**Best Practice**: 
- Create minor updates only
- For major changes, duplicate → edit → archive old → publish new

### Archiving Old Campaigns

1. Find course in management table
2. Click "Archive" button
3. Course hidden from learning dashboard
4. Progress data preserved
5. Can re-publish later if needed

### Duplicating a Course

**Use Case**: Create similar course for different campaign

1. Find course to duplicate
2. Click "Duplicate" button
3. New course created with "(Copy)" suffix
4. Status set to Draft
5. Edit the copy
6. Change title, campaign, content
7. Publish when ready

### Exporting Training Reports

1. Click "Reports" in Sales Academy
2. Review analytics
3. Click "Export Report"
4. CSV file downloads
5. Open in Excel/Sheets
6. Analyze data, create charts

### Adding Knowledge Base Articles

1. Click "Knowledge Base" button
2. Click "Add Article"
3. Enter title: "How to Handle Price Objections"
4. Enter content: "Step 1: Listen carefully..."
5. Enter tags: "sales, objections, pricing"
6. Save
7. Article added to accordion

## Tips & Best Practices

### For Admins

**Content Creation**:
- ✅ Keep courses under 60 minutes
- ✅ Use variety of content types
- ✅ Add quizzes for knowledge checks
- ✅ Include real examples
- ✅ Update regularly

**Don't**:
- ❌ Create 2-hour courses
- ❌ Use only text
- ❌ Skip testing before publishing
- ❌ Delete courses (archive instead)
- ❌ Ignore completion rates

**Course Structure** (Recommended):
1. Introduction (Video) - 5 min
2. Theory (Text) - 10 min
3. Examples (Video/Images) - 10 min
4. Practice (Text) - 10 min
5. Quiz - 5 min

**Naming Convention**:
- Campaign: "{Campaign Name} Sales Training"
- General: "{Topic} - {Level}"
- Examples:
  - "Sea Freight Sales Training"
  - "CRM Basics - Beginner"
  - "Objection Handling - Advanced"

### For Sales Members

**Learning Strategy**:
- ✅ Complete mandatory trainings first
- ✅ Review knowledge base before calls
- ✅ Take notes during training
- ✅ Practice techniques immediately
- ✅ Ask questions in team meetings

**Don't**:
- ❌ Skip mandatory trainings
- ❌ Rush through content
- ❌ Ignore quizzes
- ❌ Forget to mark sections complete
- ❌ Wait until deadline

**Time Management**:
- Block 30 minutes daily for training
- Complete 1-2 courses per week
- Review old content quarterly
- Use knowledge base during calls

## Troubleshooting

### "Training Builder not opening"
**Solution**: 
- Refresh page
- Check if you're Admin/Super Admin
- Clear browser cache

### "Cannot save course"
**Solution**:
- Check title filled
- Check category selected
- Check at least 1 section added
- Check internet connection

### "Sections not appearing"
**Solution**:
- Click "Save Section" after editing
- Check section title filled
- Refresh training builder

### "Reports showing no data"
**Solution**:
- Ensure trainings published
- Ensure users enrolled
- Wait 24 hours after first enrollment
- Refresh reports page

### "Mandatory training not blocking"
**Solution**:
- Ensure training marked as mandatory
- Ensure training published
- Clear browser cache
- Re-login

### "Course not visible to members"
**Solution**:
- Check status is "Published"
- Check course not archived
- Check campaign active (if campaign-specific)
- Refresh member dashboard

## Quick Reference

### Admin Action Buttons

| Button | Action | Result |
|--------|--------|--------|
| New Course | Create training | Opens training builder |
| Categories | Manage categories | Opens category manager |
| Knowledge Base | Manage KB | Opens KB manager |
| Reports | View analytics | Opens reports modal |

### Course Actions

| Icon | Action | Available When | Result |
|------|--------|----------------|--------|
| ✏️ Edit | Edit course | Always | Opens builder |
| ✅ Publish | Publish course | Draft | Status → Published |
| 📦 Archive | Archive course | Published | Status → Archived |
| 📄 Duplicate | Copy course | Always | Creates "(Copy)" |
| 🗑️ Delete | Delete course | Super Admin | Removes course |

### Course Status

| Status | Badge Color | Visible To | Can Edit |
|--------|-------------|------------|----------|
| Draft | Yellow | Admin only | Yes |
| Published | Green | All users | Yes (caution) |
| Archived | Gray | Admin only | Yes |

| Video | Demonstrations | Product demos, pitches |
| PDF | Reference docs | Price lists, catalogs |
| Image | Visual aids | Flowcharts, diagrams |
| Quiz | Assessments | Knowledge checks |
